# spearmint-os
Spearmint OS is a Linux Distribution based on Debian GNU/Linux

**This project is still in pre-alpha**

## Dependencies
```
live-build
```

## Building
Building is very simple, just do:
```
$ lb config --config https://github.com/DefunctLizard/spearmint-os.git::debian
```
And:
```
# lb build
```
